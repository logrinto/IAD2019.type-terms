Vertikale Typometrie

Unter vertikaler Typometrie versteht man die geometrische Konstruktion von Schriftzeichen, die sich in der lateinischen Schrift hauptsächlich an dem sogenannten Vierliniensystem orientiert.
Durch die Gestaltung der Zeichen nach diesem Prinzip, wirkt das Schriftbild einheitlich. Die einzelnen Zeichen sind proportional zueinander und der Lesefluss ist gewährleistet.

Vertikales Liniensystem

Vierliniensystem

Das Vierliniensystem der Typografie funktioniert im Prinzip ähnlich, wie das Schreiben von Musiknoten auf dem linierten Notenblatt. Durch diese vier vertikalen (Haupt-)Linien lassen sich die Buchstaben vermassen und einheitlich gestalten:

  • H-Linie
  • x-Linie
  • Schriftlinie
  • p-Linie

Weitere, relevante Linien

Das Vierliniensystem wird durch einige weitere Linien ergänzt, die essenziell für das Schriftbild sind. Zu diesen zählen:

  • Kegeloberkante
  • x-Linie
  • k-Linie
  • Kegelunterkante

Abstände

Durch die vertikalen Linien ergeben sich automatisch verschiedenste Abstände, deren Bezeichnungen ebenfalls zum typografischen Fachwissen gehören. Darunter befinden sich unter anderem:

  • Schriftgrösse
  • Versalhöhe
  • Fleisch
  • Überhang
  • Oberlänge
  • x-Höhe
  • Unterlänge

Anwendung CSS

Mit Hilfe von CSS können Objekte (z.B. ein Bild) eines HTML-Files an der Schrift ausgerichtet werden. Folgende Varianten sind möglich:

  • vertical-align: text-top;
    → Ausrichtung an Kegeloberkante
  • vertical-align: baseline;
    → Ausrichtung an Schriftlinie
  • vertical-align: text-bottom;
    → Ausrichtung an Kegelunterkante

«Sen» Schriftart

«Sen» wird als (Geo)humanistische, Serifenlose klassifiziert. Eine Humanistische Schrift zeichnet sich dadurch aus, dass sie in ihren Grundformen an das Schreiben mit der Hand erinnert (beispielsweise durch unterschiedliche Strichstärken). Sie wird von ihrem Herausgeber «Philatype» als geometrisch konstruierte, serifenlose Schrift, welche jedoch ein freundliches Aussehen vorweist, beschrieben. «Betrachten Sie sie als eine neutralere Version geometrischer Klassiker wie Avenir oder Futura mit einem humanistischen Touch. Sie ist bescheiden, einzigartig und vor allem leicht zu lesen.» – Philatype. «Sen» wurde unter der SIL Open Font License (OFL) veröffentlicht und wird kostenlos auf verschiedenen Webseiten angeboten.

Tipp

Um ein ideales Ergebnis bei Schriftmischungen zu erhalten: Vergleiche die vertikale Typometrie (vertikale Linien) der Schriften miteinander – je ähnlicher sie sind, desto besser lassen sie sich mischen.
Zudem gilt: Je höher der Kontrast der Schriftklassen, desto besser gelingt die Mischung. (z.B. Serifenlose mit Serifen mischen)
