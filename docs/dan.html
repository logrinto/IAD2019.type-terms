Versalien & Kapitälchen

Dan Büschlen • 2020 • IAD2019

Versalien

Versalien ist in der Typografie der Begriff für Grossbuchstaben. Diese werden auch Majuskel genannt. Der Begriff Versalien stammt vom lateinischen Wort «versus» ab und bedeutet so viel wie Zeile, Furche oder das Umwenden des Pfluges. Die Anwendung der Versalien wird heute in der Deutschen Sprache durch die Rechtschreibung bestimmt. So wird zum Beispiel immer der erste Buchstaben eines Satzes oder eines Nomens gross geschrieben.
Die Versalschrift wurde ursprünglich als Auszeichnungsart verwendet wie zum Beispiel auch fette und kursive Schriften. Gerade als Initiale wurden die Versalien oft verwendet.

Kapitälchen

Eine weitere Auszeichnungsart sind die Kapitälchen. Kapitälchen sind Buchstaben in der Form von Versalien, welche in der Grösse und Schriftstärke von Kleinbuchstaben dargestellt werden. Die Kapitälchen werden heute häufig verwendet um zum Beispiel Namen leise auszuzeichnen. Beinhaltet eine Schriftart keine Kapitälchen so können Textbearbeitungsprogramme Versalien skalieren. Diese werden als «falsche Kapitälchen» bezeichnet.

Verwendung mit CSS

Mit CSS können die beiden Auszeichnungsarten gesteuert werden. Mit Hilfe des text-transform Property können Versalien in einem beliebigen Element forciert werden. 

h1 {
  text-transform: uppercase;
}

In Kombination mit dem Pseudo Element ::first-letter kann eine Initiale gestaltet werden werden.

p::first-letter {
  text-transform: uppercase;
  font-size: 2em;
}

Um einen Text mit Kapitälchen darzustellen, kann das Property font-variant verwendet werden. Mit dem Wert small-caps wird der jeweils erste Buchstaben eines Wortes ignoriert, sofern dieser ein Versalbuchstaben ist. Mit all-small-caps wird auch der erste Buchstabe eines Wortes immer berücksichtigt. 

p {
  font-variant-caps: small-caps;
  font-variant-caps: all-small-caps;
}

Spectral

Spectral ist eine vielseitige Serifenschrift, die in vierzehn Schnitten und mit Kapitälchen erhältlich ist. Spectral bietet ein effizientes, schönes Design, das in erster Linie für textreiche, bildschirmgerechte Umgebungen und das Lesen von langen Texten gedacht ist.
